# Workshop: Introdução à Otimização Linear

* O workshop objetivou demonstrar como a Programação Linear (inteira) é utilizada para resolver problemas de otimização visando a melhor tomada de decisão

* Dessa forma, foi utilizada a biblioteca OR-Tools juntamente com o python para formular modelos matemáticos diversos com foco no problema de transporte

Workshop vinculado ao projeto "Otimização Geoespacial do Sistema de Abastecimento de Água do Estado do Ceará: Uma Abordagem Baseada em Ciência de Dados e Redes Complexas" em desenvolvimento pela universidade.
